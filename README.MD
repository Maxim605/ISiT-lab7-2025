# ИСиТ лабораторная 7 (2025)

## Установка

Для запуска необходим установленный node JS. Приложение адаптировано под запуск на win10 с подключенным wsl

```bash
npm install
```

## Настройка

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Отредактируйте `.env` и укажите пароль вашего sudo пользователя:

```
SUDO_PASSWORD=admin
```

## Запуск

```bash
npm start
```

Сервер запустится на порту 3001 (или указанном в `.env`).

Откройте в браузере: http://localhost:3001

Чтобы раздать localhost:3001 использовался cloudpub. 
После установки пройдите в раздел "Публтикации" -> "Добавить" -> измените дефолтный порт 8080 на 3001 и сохраните
На главной странице cloudpub вы увидете публичный адрес порта

Приятного пользования! 

P.S. с VPN tracemap.pl может работать некорректно
